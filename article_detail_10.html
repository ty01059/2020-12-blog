# 변수 ``` - 데이터를 담이놓은 공간 - 변수에 담긴 데이터 타입에 따라서 사용 방법이 다르다. - var, let, const 을(를) 사용하여 선언한다. ``` # 명명규칙 ``` * 변수 선언을 할 때는 규칙이 있다. 1) 시작문자는 영문자, _, $중 하나로 시작해야한다. 2) 예약어는 사용할 수 없다. ( switch 3) 영문자의 대소문자를 구분한다. ``` # 명명기법 ``` - 카멜 기법 - 파스칼 기법 - 언더스코어 기법 ``` # 종류 ``` 1) function-scoped - var : 다양한, 넓은 공간에서 사용 가능 2) block-scoped - const : 재선언, 재할당이 불가능하다. (한번 선언하면 같은 이름으로는 데이터변경 불가능) - let : 재할당이 가능하다. (한번 선언해도 다시 데이터 할당이 가능) ```
